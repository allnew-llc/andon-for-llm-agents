# andon-pack-ios-development

ANDON Knowledge Pack for iOS app development and App Store submission.

## What This Pack Does

When an LLM agent encounters an iOS development failure, this pack:

1. **Classifies** the failure (build error, signing issue, review rejection,
   privacy manifest problem, StoreKit error)
2. **Recommends** relevant skills with Apple framework patterns and
   App Store submission guidance

## Skills Included

| Skill | Description |
|-------|-------------|
| `apple-framework-patterns` | Common Apple framework API patterns |
| `app-review-guide` | App Store Review Guidelines reference |
| `privacy-compliance` | Privacy manifest and data collection |
| `code-signing-guide` | Code signing troubleshooting |
| `storekit-patterns` | StoreKit 2 implementation patterns |

## Requirements

None — iOS development is not a regulated domain.
