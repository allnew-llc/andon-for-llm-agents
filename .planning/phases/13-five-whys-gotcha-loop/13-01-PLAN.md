<objective>
Create the Gotcha candidate generation module that transforms Five Whys root cause + prevention text into a reviewable Gotcha candidate YAML file.

Purpose: This is the first half of the Five Whys -> Gotcha learning loop. When a Five Whys analysis completes, this module auto-generates a candidate Gotcha in gotchas/candidates/ so the learning can be captured and later promoted to the live registry.

Output: hooks/gotcha_candidate.py with generate_candidate(), tests/test_gotcha_candidate.py with TDD tests, gotchas/candidates/ directory (created on first use).
